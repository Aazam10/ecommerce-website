# Footy Store

Footy Store Ecommerce is an online store for sports lovers to buy items related to sports. It contains shoes, football club kits, and all apparels related to sports.

## Contains

- [Home](https://footy-store.netlify.app/)
- [Products](https://footy-store.netlify.app/product-list/product-list.html)
- [Cart Management](https://footy-store.netlify.app/cart/cart.html)
- [Wishlist](https://footy-store.netlify.app/wishlist/wishlist.html)
- [Login](https://footy-store.netlify.app/authorization/login/login.html)
- [Signup](https://footy-store.netlify.app/authorization/signup/signup.html)
